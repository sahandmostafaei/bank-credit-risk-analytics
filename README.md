## Overview

This project demonstrates a data-driven approach to consumer credit risk analysis using Python. It analyses borrower characteristics, loan performance, and financial indicators to better understand the factors associated with loan default.

The project follows a modular structure inspired by analytical workflows commonly used in commercial banks and financial institutions for consumer lending and credit risk management.

---

## Project Objectives

- Analyse a real-world credit risk dataset
- Explore borrower characteristics
- Perform statistical analysis
- Detect missing values
- Calculate loan default rates
- Generate descriptive statistics
- Evaluate relationships between financial variables
- Build a baseline machine learning model for credit risk prediction
- Demonstrate quantitative techniques relevant to banking and finance

## Dataset

The project uses a publicly available consumer credit risk dataset containing variables commonly used in retail lending, including:

- Borrower age
- Annual income
- Employment length
- Home ownership
- Loan amount
- Loan intent
- Loan grade
- Interest rate
- Credit history length
- Loan status (default / non-default)

The dataset is used exclusively for educational and research purposes.
