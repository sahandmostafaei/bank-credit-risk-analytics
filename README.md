# Bank Credit Risk Analytics

## Overview

This project demonstrates a data-driven approach to consumer credit risk analysis using Python. It explores borrower characteristics, loan performance, and financial indicators to understand the factors associated with loan default.

The project is inspired by analytical techniques used by commercial banks and financial institutions during credit assessment and portfolio risk management.

---

# Project Objectives

- Analyse a real-world credit risk dataset
- Explore borrower characteristics
- Perform statistical analysis
- Identify missing values
- Calculate default rates
- Produce descriptive statistics
- Prepare data for predictive modelling
- Demonstrate quantitative techniques used in banking and finance

# Dataset

The project uses a publicly available consumer credit risk dataset containing borrower financial information including:

- Age
- Income
- Home ownership
- Employment length
- Loan amount
- Loan intent
- Loan grade
- Interest rate
- Credit history length
- Loan status (Default / Non-default)

The dataset is used solely for educational and research purposes.
