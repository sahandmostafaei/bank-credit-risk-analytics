Bank Credit Risk Analytics

Overview

This project demonstrates a data-driven approach to credit risk analysis using Python. It focuses on the types of analyses performed by financial institutions when evaluating the probability that a borrower may default on a loan.

The objective is to illustrate how financial data can be explored, cleaned, visualized, and analyzed using quantitative methods commonly introduced in banking and finance programs.

Project Objectives

- Explore a credit risk dataset.
- Perform exploratory data analysis (EDA).
- Identify variables associated with loan default.
- Calculate summary statistics.
- Visualize relationships between borrower characteristics and credit outcomes.
- Build a simple baseline credit risk prediction model.
- Present findings using clear charts and tables.
